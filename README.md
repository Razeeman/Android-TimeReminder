# TimeReminder

Very simple two screen app that shows how much time is left until you reach average life expectancy. Have fun!

## Technology stack
- Java8
- Kotlin
- Androidx
- MVP
- Dagger
- RxJava
- PreferenceFragment
- Custom DatePreference
- Unit/UI tests
- Proguard.
